# 🏋️ Biblioteca de Fuerza - Hijos de la Resistencia

Aplicación web para búsqueda y gestión de ejercicios de fuerza con integración de videos de YouTube.

## 🚀 Características

- **Búsqueda inteligente** con fuzzy matching
- **Filtros avanzados** por grupo muscular, subgrupo, aplicación y material
- **Constructor de rutinas** para crear y gestionar entrenamientos
- **Integración de YouTube** con previsualización de videos
- **Copiar ejercicios** con un solo clic
- **Diseño moderno** con branding HDLR

## 📦 Instalación Local

```bash
# Clonar el repositorio
git clone <tu-repo-url>

# Instalar dependencias
pip install -r requirements.txt

# Ejecutar la aplicación
streamlit run app.py
```

## 🌐 Deploy en Streamlit Cloud

Esta aplicación está optimizada para desplegarse en [Streamlit Community Cloud](https://streamlit.io/cloud).

1. Sube este repositorio a GitHub
2. Conecta tu cuenta de Streamlit Cloud
3. Selecciona este repositorio
4. ¡Deploy automático!
